## YouTube Search
This is a simple application search and display data from the YouTube Data API

### Tools Used
- React
- React-Redux
- reactstrap
- react-toastify
- node-sass
- eslint

### Setup Requirements
- node at least v10
- npm
- Your favourite browser

### Getting Started
- Clone this repo
- Install the dependencies, `npm install`
- start the development server, `npm start`
